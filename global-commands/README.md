# Global Commands

13 slash commands for Claude Code. Install with `./setup.sh` from the kit root, or manually:

```bash
cp global-commands/*.md ~/.claude/commands/
```

Once installed, run `/commands` inside Claude Code for the full reference and installation health check.

## Commands

| Category | Commands |
|----------|----------|
| **Session Lifecycle** | `/stand-up`, `/crack-on`, `/sitrep`, `/wrap` |
| **Quality** | `/audit`, `/quick-audit` |
| **Utility** | `/pitch`, `/roast`, `/eli5`, `/shower-thought` |
| **Infrastructure** | `/sync-doe`, `/commands` |
